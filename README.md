# 💳 Subscription Tracker

A beautiful, feature-rich Flutter app for tracking and managing all your recurring subscriptions in one place. Keep tabs on your monthly spending, visualize where your money goes, and never miss a renewal date.

![Flutter](https://img.shields.io/badge/Flutter-3.0+-blue.svg)
![Dart](https://img.shields.io/badge/Dart-3.0+-blue.svg)
![License](https://img.shields.io/badge/License-MIT-green.svg)

---

## ✨ Features

| Feature | Description |
|---------|-------------|
| 📊 **Spending Dashboard** | Real-time monthly & yearly totals with a gradient hero header |
| 🥧 **Category Breakdown** | Interactive pie chart showing spending distribution by category |
| 📝 **Subscription Management** | Add, edit, delete, and toggle subscription status |
| 🏷️ **Smart Categories** | 6 built-in categories: Entertainment, Productivity, Health, Finance, Shopping, Other |
| 🔄 **Billing Cycles** | Supports Weekly, Monthly, and Yearly billing with auto-conversion |
| 📅 **Date Tracking** | Track start dates and upcoming renewal dates |
| 💾 **Local Persistence** | Data saved locally using SharedPreferences |
| 🎨 **Modern UI** | Material 3 design with smooth animations and responsive layout |

## 🚀 Getting Started

### Prerequisites

- Flutter SDK 3.0 or higher
- Dart SDK 3.0 or higher

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/subscription-tracker.git

# Navigate to project directory
cd subscription-tracker

# Install dependencies
flutter pub get

# Run the app
flutter run
